Microsoft OneNote is a computer program for free-form information gathering and multi-user collaboration. It can store notes (handwritten or typed), drawings, screen clippings and audio commentaries. Notes can be shared with other OneNote users over the Internet or a network. OneNote was available as a part of Microsoft Office. As of 2018 it is available as a free standalone application for Windows, Windows RT, and Windows Phone. It is available for macOS, iOS and Android.[5] A web-based version of OneNote is provided as part of OneDrive or Office Online; this version enables users to edit notes via a web browser.
