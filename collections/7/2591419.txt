Apple included a bundled copy of the Mac OS X Edition of World Book Encyclopedia when they made OS X the default operating system for all new computers.[4] This edition had some Mac-only features including a more intuitive user interface, Sticky Notes sharing via Bonjour technology, Trivia Challenge game, a collection of editor-approved webcams, Notepad, speech capabilities and "This Day in History", "Media Showcase" and "Librarian" widgets.
